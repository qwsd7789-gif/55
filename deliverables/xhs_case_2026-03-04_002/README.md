# 小红书发布包（案例 002）

## 主题
日进斗金💰✨｜五行喜土招财小双圈（复刻结构版）

## 目录说明
- `01_title_and_copy.md`：可直接粘贴发布的标题/正文/话题
- `02_image_generation.md`：封面与配图提示词（Suxi / Gemini 可用）
- `03_publish_manifest.json`：结构化发布参数

## 使用顺序
1. 先用 `02_image_generation.md` 生成封面候选图
2. 选 1 张作为主封面
3. 复制 `01_title_and_copy.md` 的标题与正文发布
