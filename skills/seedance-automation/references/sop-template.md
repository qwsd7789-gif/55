# Seedance 自动化 SOP 模板
## 1. 目标
一句话写清本次要生成什么视频、给谁用、成片重点是什么。
示例：
- 用 4 张美容院空间图生成 10 秒门店氛围展示视频
- 用产品图生成 8 秒电商广告短片
## 2. 输入要求
- 主题：
- 时长：
- 比例：
- 素材目录：
- 实际使用素材：
- 风格关键词：
- 是否需要声音：
## 3. 素材整理
```text
素材来源目录：
实际使用：
- 文件 1：用途
- 文件 2：用途
- 文件 3：用途
- 文件 4：用途
```
## 4. 提示词
```text
@图片1 为……参考，@图片2 为……参考，@图片3 为……参考，@图片4 为……参考。
[完整可提交提示词]
```
## 5. 提交前校对清单
- [ ] 时长正确
- [ ] 比例正确
- [ ] 引用关系正确
- [ ] 素材数量未超限
- [ ] 未使用受限真人脸素材
- [ ] 提示词中用途说明完整
- [ ] 已写明禁止项（如需要）
