核心功能
----

MCP Server 的能力范围与支持项。

[核心功能](http://listenhub.ai/docs/zh/mcp/core-capabilities#%E6%A0%B8%E5%BF%83%E5%8A%9F%E8%83%BD)
----------------------------------------------------------------------------------------------

*   **播客生成**：
    *   支持完整生成（文本 + 音频）或仅生成文本内容
    *   支持 1-2 位音色，可自定义语音
    *   提供多种生成模式：速听精华、深度探索、辩论
    *   支持先审后录生成流程（先文本后音频）
    *   支持文本和 URL 作为内容来源
    *   查询播客生成状态
    *   支持获取音频文件、标题、脚本和大纲等内容
    *   跟踪积分消耗

*   **FlowSpeech 语音生成**：
    *   将文本或 URL 等书面语内容转换为单人口语式音频
    *   支持智能模式（AI 润色后朗读）或直接模式 （原文逐字朗读）
    *   查询 FlowSpeech 生成状态
    *   支持获取音频文件、标题、脚本和大纲等内容
    *   跟踪积分消耗

*   **音色管理**：
    *   浏览可用的音色/语音库
    *   按语言和音色属性筛选
    *   获取音色 ID、姓名、语言、性别和试听音频

*   **用户账户管理**：
    *   查询订阅状态和套餐信息
    *   查看积分使用情况：月度积分、永久积分和限时积分
    *   监控可用积分总额和续订状态
    *   查看订阅开始/到期日期

*   **多种传输模式**：支持 Stdio、HTTP SSE 和 HTTP Streamable 传输方式，灵活集成

[使用示例 中文与英文播客生成示例截图。](https://listenhub.ai/docs/zh/mcp/usage-examples)[传输模式 Stdio 与 HTTP/SSE 模式说明。](https://listenhub.ai/docs/zh/mcp/transport-modes)
